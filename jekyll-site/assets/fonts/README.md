# Шрифти

Покладіть сюди ваш церковний шрифт у форматі `.woff2` або `.ttf`, наприклад:

- **Ponomar Unicode** — церковнослов'янський, безкоштовний  
  <https://github.com/typiconman/Ponomar>
- **Triodion Unicode** — слов'янський академічний  
  <https://irmologion.ru/fonts/>

Інструкція з підключення — у головному `README.md` (розділ «Як додати справжній церковний шрифт»).
